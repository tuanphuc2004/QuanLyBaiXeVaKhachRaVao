## Backend - Visitor & Vehicle Access Control

Thư mục này chứa mã nguồn backend ASP.NET Core Web API cho hệ thống "Hệ thống Quản lý Người & Phương tiện Ra vào Công ty".

- Project chính: `VisitorAccessControl.Api`
- Công nghệ:
  - ASP.NET Core Web API (.NET 8)
  - Entity Framework Core (SQL Server)
  - JWT Authentication
  - SignalR (Dashboard realtime)
  - Clean Architecture cơ bản (Controller - Service - Repository)
